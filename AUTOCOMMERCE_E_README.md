# AutoCommerce Enterprise README

Package de livraison entreprise pour AutoCommerce V25.

## Objectif
Fournir un bundle exploitable par une équipe technique ou un comité de mise en production, avec :
- code backend et frontend
- stratégie de déploiement
- contrôles qualité et sécurité
- documentation B2B et monitoring
- dossier de go/no-go

## Mode d’emploi rapide
1. Lire `README.md`
2. Lire `docs/deployment-strategy.md`
3. Vérifier `docs/go-nogo-v25.md`
4. Exécuter le preflight release
5. Réaliser le round-trip manuel entreprise
6. Utiliser `docs/go-nogo-v25-final.md` comme document de décision

## Wording officiel recommandé
Tant que le round-trip final n’est pas exécuté et signé, utiliser la formule :
**“Ready for enterprise release validation, pending final manual round-trip sign-off.”**
